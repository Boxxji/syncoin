# Contribuer à SynCoin 🌱

Merci de vouloir contribuer à SynCoin ! Ce projet est né d'une amitié et grandi pour le bien commun.

## Comment contribuer

### Signaler un bug
Ouvrez une issue sur GitHub en décrivant :
- Le comportement attendu
- Le comportement observé
- Les étapes pour reproduire
- Votre environnement (OS, version Python)

### Suggérer une amélioration
Créez une issue avec le label `enhancement` et décrivez :
- Le problème que vous voulez résoudre
- Votre solution proposée
- Des alternatives envisagées

### Soumettre du code

1. Forkez le projet
2. Créez une branche (`git checkout -b feature/ma-fonctionnalite`)
3. Commitez vos changements (`git commit -m 'Ajoute ma fonctionnalité'`)
4. Poussez (`git push origin feature/ma-fonctionnalite`)
5. Ouvrez une Pull Request

### Règles de contribution

- **100% non-commercial** : Aucune utilisation commerciale du code
- **Pour le bien commun** : Les contributions doivent servir le bien collectif
- **Respectueux** : Suivez le Code de Conduite
- **Testé** : Assurez-vous que le code fonctionne avant de soumettre

### Architecture du projet

```
syncoin_node.py    → Nœud P2P principal
mobile/            → App Flutter (iOS/Android)
ios/               → App Swift native
contracts/         → Smart contracts Solana
docs/              → Documentation
```
